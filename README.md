# Election Contribution Visualization using Plotly

## Project Overview

This project explores political contribution data from the 2018 Election Cycle using multiple interactive visualizations created with Plotly in Python.

The objective was to represent the relationships between sectors and political recipients through different visualization techniques and compare how each chart communicates insights from the same dataset.

## Visualizations Included

* Sankey Diagram
* Sunburst Chart
* Treemap
* Stacked Bar Chart
* Heatmap
* Pie Chart

## Technologies Used

* Python
* Pandas
* Plotly
* Google Colab

## Dataset

The dataset contains contribution amounts from various sectors such as:

* Lawyers & Lobbyists
* Finance/Insurance/Real Estate
* Health
* Defense
* Energy & Natural Resources
* Transportation
* Agribusiness

Recipients include:

* Democrats
* Republicans
* Outside Spending Groups

## Key Insights

* Finance/Insurance/Real Estate contributed significantly across recipient groups.
* Ideology/Single-Issue organizations showed strong support for Outside Spending Groups.
* Contribution patterns vary substantially across sectors and political recipients.
* Different visualization techniques highlight different aspects of the same data.

## Learning Outcomes

Through this project, I learned:

* Hierarchical data visualization using Sunburst and Treemap charts
* Flow visualization using Sankey diagrams
* Comparative analysis using Stacked Bar Charts and Heatmaps
* Interactive visualization development with Plotly
* Data manipulation and transformation using Pandas
